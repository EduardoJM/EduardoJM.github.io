## Running on Localhost

1. clone the repository:

```
git clone https://github.com/EduardoJM/EduardoJM.github.io.git
cd EduardoJM.github.io
```

2. install the dependencies:

```
yarn
```

or

```
npm install
```

3. run the local development server:

```
yarn start
```

or

```
npm start
```
