Análise de Variância (ANOVA)

Para verificar se o modelo de regressão linear é significativo.

SST=SSR+SSESS_T = SS_R + SS_ESST​=SSR​+SSE​ ∑(y−yˉ)2=∑(y^−yˉ)2+∑(y−y^)2\sum (y-\bar{y})^2 = \sum (\hat{y} - \bar{y})^2 + \sum (y-\hat{y})^2∑(y−yˉ​)2=∑(y^​−yˉ​)2+∑(y−y^​)2
  • SSTSS_TSST​ (também SQTSQ_TSQT​) é Soma de Quadrados Total
  • SSRSS_RSSR​ (também SQRSQ_RSQR​) é Soma de Quadrados da Regressão
  • SSESS_ESSE​ (também SQESQ_ESQE​) é Soma de Quadrados do Erro (Residual)
